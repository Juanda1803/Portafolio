Proyectos

VideoCode

VideoCode es una aplicación donde permite ver las últimas tendencias y los mejores videos de platzi relacionados con el código y la programación. Este proyecto básicamente fue inspirado para aprender HTML, CSS y JavaScript junto con las librerías de React, React Router y Redux. VideoCode permite hacer un registro e inicio de sesion, donde cada usuario guardara y podra mirar los mejores videos de platzi.

AgrodatAI

Foro-AgrodatAI es un foro interactivo dividido en 4 partes. Foro, Crear comunidades, Mis comunidades y Comunidades destacadas. La idea de este proyecto es permitir que los usuarios de AgrodatAI puedan preguntar a la comunidad temas sobre las diferetes actividades como Agricola, Bovino, Porcino, Avicola y Acuicola.

PlatziBadges

Es una plataforma para llevar el registro de asistentes en la Platzi Conf, donde permitira a cada usuario registrado poder ingresar a la conferencia, ingresando la información personal. También agregue una página en donde permite listar todos los participantes y poderlos filtrar en un buscador por el nombre. Se manejaron muchos temas de React js, manejamos estados, componentes, aplicación de rutas, ciclo de vida de un componente, peticiones, hooks etc.

Petgram

Petgram es una aplicación inspirada en instagram con el propósito de crear algo diferente y nuevo para los amantes a los animales. Cuenta con una con un amplio stack de fotos de los diferentes tipos de animales como perros, gatos , conejos ,peces .e.t.c. Petgram permitirá al usuario registrarse e iniciar sesión y así tener un acceso exclusivo a todas las funcionalidades de la aplicación como dar like, guardar fotos favoritas, cerrar sesión .e.t.c. Principalmente se manejo la tecnología de React usado componentes y hooks en la aplicación.
