FunDog

Este es un proyecto de una página web que se conecta con PayPal para realizar el proceso de pago de los productos ofrecidos. Realice la creación de un producto digital dándole diseño y maquetación. Esta página web contiene la descripción del producto y un formulario donde te piden el nombre, correo y teléfono celular. Al darle clic confirmar pago te redirecciona a una página de PayPal mostrando cuanto debe pagar por el producto. Este dinero pasa a la cuenta de PayPal del vendedor, con la información recibida por el formulario.
